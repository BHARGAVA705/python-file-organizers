## Description
This project is a Python automation script that organizes files in a given folder
based on their file type. It uses the os module to manage files and folders,
includes exception handling, logging, and user input.

This project is developed as part of an internship task.

---

## Features
- Organizes files automatically
- Uses os module for file operations
- Handles errors using try-except
- Generates log file for tracking operations
- Accepts user input for folder path

---

## Technologies Used
- Python
- os module
- logging module

---

## How to Run the Project
1. Install Python
2. Open the script in PyCharm
3. Run `file_organizer.py`
4. Enter the folder path when prompted

---

## Sample Input
C:\Users\SUPER\OneDrive\Pictures\spidy.jpg

---

## Sample Output
- Images moved to `Images` folder
- PDFs moved to `PDFs` folder
- Text files moved to `TextFiles`
- Other files moved to `Others`

---

## Logs
A log file named `file_organizer.log` is created to track all operations.
